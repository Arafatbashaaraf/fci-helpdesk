## Change reporting manager

Transfer reporting responsibility when team structures change.

## Steps

### Open employee profile

From the directory, search the employee and open their profile.

### Select manager field

Choose the reporting manager section and click edit.

### Assign new manager

Search and select the new manager, then save changes.

### Confirm hierarchy update

Validate that org chart and approvals reflect the new reporting line.

> [!success] Communicate manager changes to payroll and attendance approvers.
