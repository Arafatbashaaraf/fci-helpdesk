## Why address settings matter

Accurate address details improve trust, communication, and compliance.

## Configure addresses

### Select address module

Go to Company Profile and open the address section.

### Edit address details

Use the edit action to update registered office, corporate office, and any custom office address.

### Bulk upload

Import address entries using Excel when managing multiple locations.

> [!success] Keep legal and mailing addresses up to date for statutory filings.
