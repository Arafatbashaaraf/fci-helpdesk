## Configure designations

Set clear roles and levels to maintain a professional organizational hierarchy.

## Steps

### Select designation section

Open Company Profile and go to Designation.

### Add designation title

Create designation records with title and optional grade details.

### Edit and save

Update existing designations as needed and save to apply.

> [!warning] Standardize naming conventions before scaling to avoid duplicate titles.
