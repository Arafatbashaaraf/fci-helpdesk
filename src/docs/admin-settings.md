## Admin controls

Manage who can edit company profile modules and approve key workflows.

## Setup flow

### Add admins

Assign users as admins based on role and responsibility.

### Configure permissions

Grant module-level access for profile, policy, payroll, and employee data.

### Verify access

Test with a role-based checklist before going live.

```json
{
  "role": "Company Admin",
  "canEditProfile": true,
  "canManagePolicies": true
}
```
